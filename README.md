# Phrmacy Management System - Java
This my first Java desktop application developed using NetBeans.
The application is designed to manage medicines in a pharmacy using object-oriented-programming (OOP) principles and a graphic interface (GUI).
## Features
- Add medicine

- Delete medicine

- Search medicine by name and type

- Display all medicine 

- supports different medicine types :

  - Tablet

  - Syrup

  - Ointment

  - Medical Device

  ## Object-Oriented-Programming Concepts Used
- Classes and Objects

- Encapsulation (private attributes with getters and setters)

- Heritage (different medicine types)

- polymorphism

- Constructors

- Method Overriding

- Separation between application logic and graphical interface


## Technologies Used
- Java

- Swing GUI

- NetBeans IDE

- File handling (for data storage)
